Модуль Turtle.

Модуль Turtle - это графическая библиотека, встроенная в язык программирования Python. Она работает с растровой(пиксельной) графикой.

План:

  1. Дизайн Черепашки
  2. Движение Черепашки
  3. Работа Черепашки

1. Дизайн Черепашки

1.1 Форма Черепашки

Для изменения формы Черепашки мы используем функцию t.shape()

t.shape('arrow') - стрела
t.shape('circle') - кружок
t.shape('square') - квадрат
t.shape('triangle') - треугольник
t.shape('turtle') - черепашка
t.shape('classic') - классическая форма

1.2 Прочее

pencolor(color) - цвет пера, которым рисует Черепашка
speed() - скорость Черепашки
pensize() - размер пера Черепашки
hideturtle() - спрятать черепашку
exitonclick() - сохранить рисунок

2. Движение Черепашки

2.1 Начало

Для начала нам надо подключить модуль Turtle. Нужно написать: from turtle import *

2.2 Настройка пера

pendown - опустить перо
penup - поднять перо

2.3 Повороты и перемещение

forward() - Черепашка двигается вперёд на ... пикселей
goto() - перемещение Черепашки на определенные координаты
left() - поворот налево на ... градусов
right() - поворот направо на ... градусов

3. Работа Черепашки

Черепашка управляется через координатную систему экрана:

  • Экран черепашки имеет центр с координатами (0, 0).
  • Положительные значения X идут вправо, отрицательные — влево.
  • Положительные значения Y идут вверх, отрицательные — вниз.
